Serenly – Mental Health Support Web Application

Serenly is a web-based mental health support application designed to assist individuals dealing with depression and emotional stress. The application focuses on self-care, emotional awareness, and habit building through simple and accessible digital tools.

Problem Statement

Depression affects mental well-being, daily productivity, and quality of life. Many individuals lack access to simple platforms that help them track emotions, reflect on thoughts, and maintain healthy routines in a safe and supportive environment.

Solution Overview

Serenly provides a centralized web platform that helps users manage mental well-being by tracking moods, journaling thoughts, setting goals, and accessing educational mental health resources.

Key Features
	•	Mood tracking to monitor emotional patterns
	•	Journaling for self-reflection and emotional expression
	•	Goal setting for mental wellness improvement
	•	Medication reminder functionality
	•	Curated mental health educational resources
	•	Multi-language support (English and Tamil)
	•	Responsive design for mobile and desktop

Technology Stack
	•	HTML
	•	CSS
	•	JavaScript
	•	Figma (UI/UX Design)
	•	GitHub Pages (Deployment)
